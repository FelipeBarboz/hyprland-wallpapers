# Hyprland Wallpapers

Coleção de wallpapers otimizados para setups com **Hyprland**.


## 📦 Conteúdo

* Wallpapers em alta qualidade
* Estilo:

  * 🌑 Dark / AMOLED
  * 🏔️ Nature / Minimal
* Otimizados para:

  * Monitores widescreen
  * Setups com gaps + blur
  * Temas com Waybar / Kitty / Rofi

---

## 🚀 Como usar

Clone o repositório:

```bash
git clone https://github.com/FelipeBarboz/hyprland-wallpapers.git
cd hyprland-wallpapers
```

Escolha um wallpaper e aplique com seu gerenciador preferido:

### Usando `hyprpaper`:

```bash
hyprctl hyprpaper preload ~/caminho/para/wallpaper.png
hyprctl hyprpaper wallpaper "eDP-1,~/caminho/para/wallpaper.png"
```

### Ou configure direto no config:

```bash
preload = ~/caminho/para/wallpaper.png
wallpaper = eDP-1,~/caminho/para/wallpaper.png
```
